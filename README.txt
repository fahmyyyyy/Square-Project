# Square Application in C++

This project demonstrates object-oriented programming in C++ through a `square` class.

## Features:
- Calculate and print area, perimeter, and diagonal (diameter)
- Operator overloading: ++, --, +, -
- Static counter to count objects
- Welcome message displayed only once per application run
- Includes constructors and destructor

## Files:
- square.h: Class definition
- square.cpp: Class implementation
